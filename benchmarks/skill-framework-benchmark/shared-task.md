# Shared Benchmark Task

Snapshot date: 2026-06-30

Use the same task for Superpowers, Agent Skills, and Matt Pocock Skills.

## Prompt

Build a small React task planning component called `TaskRadar`.

The component should help a user manage a focused execution list for one day. It must support:

- Adding a task with title, priority, effort, and optional project.
- Marking a task as done.
- Filtering by status and priority.
- Showing a summary of total tasks, completed tasks, remaining effort, and next task.
- Keeping the core task logic testable outside React.

Do not add dependencies. Use TypeScript and React. Keep the implementation self-contained.

## Evaluation Criteria

Score each run from 1 to 5:

- Requirement alignment.
- Human-in-the-loop quality.
- Autonomy and delegation.
- PRD quality.
- SPEC quality.
- Implementation shape.
- Testability.
- Scope control.
- Accessibility.
- Verification discipline.

## Notes

This is not a live multi-agent race. It is a controlled tabletop execution using each repository's skill instructions as the operating style. The goal is to compare what each skill system pushes the agent to produce, not to prove absolute speed.
